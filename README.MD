# Smart Glasses for Visually Impaired Person

This project was developed during a Hackathon to assist visually impaired people using Raspberry Pi and Python.

## Features
- Obstacle Detection
- Object Detection
- Currency Detection
- Voice Alert System
- Real-time Assistance

## Technologies Used
- Python
- Raspberry Pi
- Computer Vision
- Text-to-Speech
- Sensors

## Modules
- obstacle.py → Obstacle detection
- object_detect.py → Object recognition
- currency_detect.py → Currency identification
- tts.py → Voice output

## How It Works
The system detects surroundings using sensors and camera modules and provides voice feedback to help visually impaired users navigate safely.
